--- --- TCA — Terminal Colors Architecture
Terminal Colors Architecture

Your theme.
Every terminal. Every CLI. Every TUI.

TCA is a structured, portable format for terminal and cli/tui color themes. Define your palette once in TOML - export to any terminal, editor, or UI framework.

cargo install tca
GitHub Specification Contribute a Theme
Who is This For?

Application Developers, Theme Developers, and End Users

Let's make color themes easier to use for everyone.

💻

Application Developers

Easily integrate color themes into your terminal application with library support for Python/Textual, Rust/Ratatui, and Go/Lipgloss support.

🎨

Theme Developers

Create your theme your way with clear, simple guardrails to help your theme be used the right way.

🛼

Theme Users

Install a theme file once, and then every TCA-aware application can use it. Export to Base16/24 for an expanded ecosystem of tools.

Current Status

DRAFT

TCA is currently a draft in review, but already supports most basic features including language libraries, theme validators, and import/export.

Ecosystem

Libraries & Tools

TCA themes work across languages and frameworks. Pick the library for your stack.

🎨

tca-themes

Ready-to-use theme collection and the canonical JSON schema for validation.

🦀

tca-rust

Core Rust crates: CLI tool, type definitions, theme loader, and Ratatui widgets for colorful terminal applications.

🐹

tca-go

Go library with Lipgloss integration for colorful terminal applications.

🐍

tca-python

Python library with Textual integration for colorful terminal applications.

Quick Start

Get Going Fast

Install the CLI, validate a theme, and export to your terminal in under a minute.

CLI

# Install
              $ cargo install tca

              # Validate a theme
              $ tca validate dracula

              # List installed themes
              $ tca list
